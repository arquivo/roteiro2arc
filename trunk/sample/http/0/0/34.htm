About the Digital Tradition Folk Song Server

The Digital Tradition is a data base containing the words and music to thousands of folk songs. These songs have been collected over the last six years by Dick Greenhaus and friends. Originally distributed as a free PC DOS program with full text search and audio playback, the Digital Tradition is now also available for Macintosh computers, as plain ASCII text (e.g. for UNIX systems), and most recently via a World-Wide Web Internet server.

The World-Wide Web Digital Tradition Server

In October 1993, Steve Putz created a World-Wide Web Server (on pubweb.parc.xerox.com) that allows users with an Internet connection to search for and display songs interactively using any of the readily available WWW browsers. The Web server can even provide audio of the song tunes for some systems. A paper describing this Web server was presented at the First International WWW Conference in Geneva, May 1994:
  • Interactive Information Services Using World-Wide Web Hypertext (HTML)
For some software and information about using tunes from the Digital Tradition, see:
