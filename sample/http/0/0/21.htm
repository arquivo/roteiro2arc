Como contribuir para esta página

Esta página não pretende ser uma lista exaustiva de todos os projectos envolvendo tecnologia X.500. O que se pretende é dar uma ideia geral das diferentes aplicaçães da tecnologia X.500 e ao mesmo tempo incentivar a divulgação de projectos em curso criando um local onde possam ser referenciados.

Se estiver envolvido em algum projecto baseado em tecnologia X.500 e estiver interessado em partilhar os resultados e experiências resultantes envie para x500@rccn.net uma mensagem a solicitar que seja criada nesta página uma referência para o seu trabalho.

Embora a informação a disponibilizar possa variar bastante, o ideal (mas não obrigatório) é a criação de uma página HTML com os seguintes elementos:

Identificação do projecto, organização, pessoa(s) a contactar, data de realização, breve descrição dos objectivos, âmbito e principais pontos de interesse do projecto, referências para informações adicionais.
