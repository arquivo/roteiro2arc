Acesso ao X.500 por mail

Este documento faz parte do servidor WWW do Serviço X.500 da RCCN

Consulta de informação na Directoria via mail

Como usar o correio electrónico para aceder ao serviço de Directoria?

Enviando uma mensagem de correio electrónico para o endereço: 

        x500-query@dsa.fccn.pt
você pode aceder ao serviço de directoria da RCCN. Envie nessa mensagem um comando com o seguinte formato: 
	find nome [,departamento],organizacao,pais

	Ex: find Joaquim Macedo,di,um,pt
algures no campo "subject" ou no corpo da mensagem. Em resposta recebera uma mensagem com a informação pretendida. No exemplo anterior receberia uma mensagem com informação sobre todos os Joaquim Macedo da Universidade do Minho em Portugal. Se só for encontrada uma pessoa receberá informação detalhada sobre ela. Se forem encontradas varias pessoas receberá uma lista que podera servir para refinar a pergunta numa proxima utilização.

Note que este serviço é inteiramente automático, nenhuma pessoa responde a mensagens para este endereço. Se pretender colocar questões ou fazer comentários deve enviar mensagens para o seguinte endereço: 

	x500@rccn.net
